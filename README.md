# Pycalc

Pure-python command-line calculator.

## Requirements
Python 3.6 and later.

## Installation
Clone this repositoty
```
git clone https://github.com/ssezmor/pycalc.git
```

Install by pip
```
pip install pycalc
```

## How to use
It provides the following interface
```
pycalc "EXPRESSION"
```
